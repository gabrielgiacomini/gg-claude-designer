Claude Design is a web-based tool at claude.ai/design that produces shareable, exportable visual deliverables through conversational prompting. It generates polished UI mockups, interactive prototypes, slide decks, and microsites rather than one-off chat artifacts. Product managers use it to create stakeholder-ready visuals, investor-facing decks, and engineering handoffs without writing code.

The tool operates by composing prompts that name the goal, layout, content, and audience. Before prompting, brand assets and product facts should be gathered—output fidelity depends entirely on this preparation. The default behavior produces multiple stylistic variations on a single canvas; users must explicitly request one design if that is their intent. Four iteration modes exist: chat for structural changes, comments for targeted feedback, live controls for adjusting values, and direct editing for property changes. Exports include PDF, PPTX, standalone HTML, Canva integration, and direct handoff to Claude Code for engineering implementation.

Access requires a Pro, Max, Team, or Enterprise plan. Enterprise organizations must have the feature enabled by an admin under Organization settings. The tool has its own usage allowance separate from standard chat and Claude Code limits. Safety rules prohibit agents from changing sharing permissions or triggering file downloads without explicit user approval—product managers confirm these actions themselves. Known limitations include no Figma export (input only), no audit logs for admins, and no data residency controls, which matters when uploading sensitive brand assets.
